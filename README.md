# Laptop Price Prediction

This project predicts laptop prices using machine learning models (Random Forest, XGBoost, Stacking Regressor) based on specifications like CPU, RAM, SSD, screen, etc.

## How to Use

1. Open the notebook `Laptop_Price_Prediction.ipynb` in Colab.
2. Run the cells step by step.
3. Predict prices for new laptops using the provided pipeline.

## Files
- `df.pkl` : cleaned dataset
- `pipe.pkl` : trained pipeline
